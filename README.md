# acLife

acLife is a personal organizer app for managing tasks, schedules, and personal notes in one place.

## Project Status

Early development. Not ready for regular use.

## Features

- **Calendar**: Create events and manage your schedule.
- **Journal**: Record notes, thoughts, and reflections.
- **Privacy-focused**: End-to-end encryption and minimal data collection.

## Technology

- **Web Client**: React + Vite + TypeScript
- **Server**: Go, REST API, Stripe integration, secure authentication (SRP)
