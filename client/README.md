# acLife Client

Web client for acLife built with Vite, TypeScript, and React.

It uses Shadcn UI components and Tailwind CSS for styling.

## Installation

```bash
git clone https://github.com/atar4xis/acLife.git
cd acLife/client
pnpm install
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
